Медичний КАНАБІС

Медична марихуана - це термін для похідних рослин Cannabis sativa, які використовуються для полегшення симптомів викликаних певними захворюваннями.

Марихуана містить в собі понад 200 активних компонентів, основні два : тетрагідроканабіол та канабідіол.

ЯК ДОПОМПГПЄ КАНАБІС?

  • зменшує тривогу;
  • зменшує запалення та знімає біль;
  • контролює нудоту та блювоту, викликані хіміотерапією;
  • вбиває ракові клітини та сповільнює ріст пухлин;
  • стимулює апетит;
  • має протисудомний та релаксуючий ефект;
  • долає стрес;
  • покращує якість сну.

*Закон від 16.02.2024 “Про внесення змін додеяких законів України щодо державного регулювання обігу рослин роду коноплі (Cannabis) № 3528-IX.

*Закон передбачає, що всі засоби на основі канабісу будуть маркуватися унікальним електронним ідентифікатором.
